# Kashidasu 図書館管理システム - 使い方ガイド

## 🚀 クイックスタート

### 1. サーバー起動
```bash
# プロジェクトディレクトリに移動
cd Kashidasu

# 依存パッケージをインストール
npm install

# サーバー証明書を発行（初回のみ）
sh ./ca/init.sh

# サーバー起動
npm start
```

### 2. ブラウザでアクセス
```
https://localhost
```

### 3. ログイン
- **管理者ID**: `0123456789`
- **パスワード**: `password`

## 📖 基本操作

### 📚 書籍管理

#### 書籍一覧の表示
1. メニューから「書籍一覧」をクリック
2. 全書籍の一覧が表示される
3. ページングで30件ずつ表示

#### 書籍検索
1. 「書籍一覧」ページの検索ボックスにキーワードを入力
2. タイトル、著者、ISBNで検索可能
3. リアルタイムで検索結果が表示される

#### 期限切れ書籍の確認
1. 「期限切れの本だけを表示」ボタンをクリック
2. 返却期限が過ぎた書籍のみが一覧表示される
3. 通常表示に戻すには「すべての本を表示」ボタンをクリック

#### 書籍情報の編集
1. 書籍一覧で編集したい書籍の「編集」ボタンをクリック
2. 書籍情報がフォームに表示される
3. ⚠️ **注意**: 貸出中の書籍は編集できません
4. 情報を編集して「編集」ボタンをクリック

#### 書籍の削除
1. 書籍一覧で削除したい書籍の「編集」ボタンをクリック
2. 「削除」ボタンをクリック
3. 確認ダイアログで「OK」をクリック

### 📦 書籍登録

#### 個別登録（バーコード方式）
1. メニューから「個別登録」をクリック
2. カメラの許可を求められたら「許可」をクリック
3. 書籍のバーコードをカメラでスキャン
4. 自動でISBNが読み取られ、書籍情報が登録される

#### 一括登録（CSVファイル）
1. メニューから「一括登録」をクリック
2. 「CSVファイルでの上書き」タブを選択
3. CSVファイルを選択またはドラッグ&ドロップ
4. 「登録」ボタンをクリック
5. ⚠️ **注意**: 既存の書籍データがすべて削除されます

#### 一括登録（テキスト入力）
1. メニューから「一括登録」をクリック
2. 「テキストボックスでの追加」タブを選択
3. 以下の形式で書籍情報を入力：

```
ISBN,タイトル,著者
9784873117720,情報アーキテクチャ,Louis Rosenfeld
9784167159005,吾輩は猫である,夏目漱石
```

4. 「登録」ボタンをクリック
5. 既存の書籍は保持されたまま、新しい書籍が追加されます

### 📚 貸出・返却

#### バーコードでの貸出
1. メニューから「バーコード読取」をクリック
2. カメラの許可を求められたら「許可」をクリック
3. まず利用者カードのバーコード（10桁）をスキャン
4. 次に書籍のバーコード（ISBN）をスキャン
5. 貸出処理が自動で実行される

#### 貸出期限について
- **管理者ユーザー**: 貸出日から14日後
- **一般ユーザー**: 貸出日から7日後
- 期限は書籍一覧に `(貸出日) -> (返却期限日)` の形式で表示

#### 返却処理
1. メニューから「バーコード読取」をクリック
2. 利用者カードのバーコードをスキャン
3. 返却する書籍のバーコードをスキャン
4. 返却処理が自動で実行される

### 👤 利用者カード

#### カードの生成
1. メニューから「カード生成」をクリック
2. 利用者情報を入力：
   - ID: 10桁の数字
   - 学年: 例: 1年生
   - クラス: 例: A組
   - 出席番号: 例: 01
3. 「カード生成」ボタンをクリック
4. PDF形式の利用者カードが生成される

## 🔧 便利な機能

### 📊 状態表示の意味
- **空き**: 書籍が利用可能
- **貸出中**: 現在誰かに貸出中
- **返却済み**: 返却処理が完了

### 📅 期限表示の読み方
```
(26/03/09) -> (26/03/16)
```
- 左側: 貸出日（年/月/日の2桁表示）
- 右側: 返却期限日
- 上記例: 2026年3月9日に貸出、期限は3月16日

### 🔍 検索のコツ
- **部分一致**: タイトルや著者の一部でも検索可能
- **ISBN検索**: 完全なISBN番号で検索
- **大文字・小文字**: 区別せず検索

### ⚠️ 注意点
- **貸出中の書籍**: 編集・削除できません
- **CSV上書き**: 既存データがすべて削除されます
- **バーコード読取**: カメラの許可が必要です
- **HTTPS接続**: 必ずHTTPSでアクセスしてください

## 🆘 トラブルシューティング

### サーバーが起動しない
```bash
# Node.jsのバージョン確認
node --version  # 22.x以降が必要

# ポート確認
netstat -an | findstr :443  # Windows
lsof -i :443                # macOS/Linux
```

### バーコードが読み取れない
- カメラの権限を確認
- 十分な光量を確保
- バーコードの距離を調整
- ブラウザを最新版に更新

### APIエラーが表示される
- HTTPSでアクセスしているか確認
- 環境変数が正しく設定されているか確認
- ネットワーク接続を確認

### 書籍が表示されない
- 書籍データが存在するか確認 (`repository/book.json`)
- ブラウザのキャッシュをクリア
- ページを再読み込み

## 📱 モバイルでの利用

### スマートフォンからのアクセス
1. 同じWi-Fiネットワークに接続
2. サーバーPCのIPアドレスを確認:
   ```bash
   # Windows
   ipconfig
   
   # macOS/Linux  
   ifconfig
   ```
3. ブラウザで `https://[IPアドレス]` にアクセス
4. 証明書の警告を無視してアクセス

### モバイルでのバーコード読取
- カメラアプリの権限を許可
- 明るい場所で使用
- バーコードを画面いっぱいに表示
